### Prerequisites
- **Tool:** Robot Framework
- **IDE:** Pycharm IDE
- **Framework:** Robot

### How to run the project
1. Clone the repo
2. Open cmd in the root folder
3. Give the following commands :
 ```
 robot Requests
 ```
